# ffstats
Alternative status page for freifunk routers
Tested on TP-Link TL-WR841ND (v9) running with gluon-v2015.1.1.

## Installation
1. Clone (or download) repository
 ````bash
 git clone https://github.com/shiftpi/ffstats.git ./ffstats
 ````
2. Run ./install.sh
